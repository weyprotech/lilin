LILIN IP Video Software Alliance Program (SAP)

LILIN are dedicated to becoming one of the world's leading providers of IP Video solutions. LILIN is committed to innovation and technological excellence. We focus on delivering products that help our customers benefit from adopting the latest IP technology. With this objective, LILIN SAP program helps software partners providing system solution to our customers. Software partners can benefit from sharing the latest IP camera technologies and direct technical support for fast product delivering.

We provide:

- Co-marketing opportunity
- Direct software developer support
- Free HTTP-based and LILIN ONVIF-based SDK download
- Free on-line ONVIF IP camera testing and integration services
- LILIN Universal ActiveX control SDK for fast RTSP video development

JOIN SAP

Please register it at LILIN's web site via
E-mail: sap@meritlilin.com.tw
Once the SAP program is activated, your company information will be listed at LILIN's web site for co-marketing purpose.

To join SAP program, please prepare:
• Company name
- Type of business
• Short description of your company
- Company logo
- Contact information
- Company website
